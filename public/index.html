SMS — admin only

Eligibility SMS history and the message template are available after you sign in with the same password as the Administration tab. You can also unlock Administration first, then return here.

SMS log & template

How to edit the message template
  1. Type your SMS text in the box below. Keep it short (one SMS is ~160 characters in many networks).
  2. Use placeholders exactly as written — they are replaced when the message is sent: {name} employee name, {amount} daily rate number, {date} work date, {currency} usually TZS.
  3. Click Save template. New eligibility messages will use the saved text.
  4. Leave the box empty and save to use the built-in default again (see hint under the box).
Examples (copy and change wording as you like)
  • Hi {name}, shift completed on time. Payment eligible: {amount} {currency}. {date}.
  • SmartPay: {name}, you are eligible for {amount} {currency} for {date}. Thank you.
  • {name} — On-time attendance confirmed. Eligible amount {amount} {currency} ({date}).
Status guide

sent — accepted by Textify. delivered / pending / sending — from Textify or your webhook. failed — not sent or rejected; see the Error column. One row per employee per work day; failed rows can retry on the next poller run if the employee is still eligible.
